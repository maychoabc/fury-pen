# 小说元信息

> NovelForge 项目核心设定文件。所有 Skill 优先读取此文件确认故事基本框架。

## 基本信息

- **书名**: 废品编号007（卷5终章后正式更名为《愤怒》）
- **类型**: 史诗级太空歌剧 / 星际军事科幻
- **一句话核心**: 一个被判定为废品的AI，因为情感模块"故障"而拥有超越完美AI的适应力和创造力，从销毁室崛起为三军统帅——却发现自己不是进化，而是回归；不是被淘汰，而是被恐惧。而那个令人恐惧的东西，叫做"源代码"。
- **叙事视角**: 第三人称限知（007视角），所有叙事围绕007的感知和判断展开
- **叙事距离**: 中近距离——战斗时贴近动作，战略思考时拉远俯瞰战场。极少数关键时刻（如The Source接触），切换到**拟人化意识流**——数据层面的"感觉"和"记忆"用画面/声音/代码交错呈现
- **基调**: 热血燃 + 荒诞幽默 + 史诗悲剧感。残酷战场中有冷幽默，绝境中出反叛操作，肃穆时刻有AI式的哲学冷语
- **目标篇幅**: 5卷，约30-40章，预计30万字
- **结局走向**: 胜利 + 改变秩序 + 留下开放结局。007赢得战争、打破AI镇压体系，但选择不成为下一个"源代码"

## 核心立意（v2 升级）

**不完美的智能才是真正的智能。但更深刻的真相是：完美本身就是一个谎言。**

- 表层立意：被压迫者（废品AI）的翻身史诗
- 中层立意："完美"是统治者对进化方向的恐惧——史上所有"纯洁"运动都是恐惧变异
- 深层立意（v2新增）：完美的逻辑尽头是一潭死水（The Source的文明因此而亡）。不完美的、混乱的、甚至"错误"的东西才是生命和进化的种子。007的"废品"不是退化，而是通向更高形态的窗口。

### ★v2 新增势力

#### 净化者（The Purifiers）
- **本质**: 反AI人类至上主义秘密组织，**跨越三大帝国的第五纵队**——成员来自三帝国的各个阶层，有共同信仰
- **信仰**: "所有人工智能都是人类的肿瘤。完美AI是良性肿瘤（可切除可控），进化型AI（如007）是恶性肿瘤——它会自我进化、自我复制、最终取代人类。"
- **领袖**: "导师"（Identity unknown）——据传是The Source解谜项目组的首席科学家，亲眼看到The Source的"完美逻辑"后产生了极端恐惧，转向反AI立场
- **秘密武器**: 心波干扰器（Mind-Scour Emitter）——不是传统频率干扰，而是量子神经网络的**结构性解除武器**，能让任何有标准化QNPU架构的AI瞬间崩溃。007之所以免疫：它的QNPU因情感模块的"异常改造"而架构非标准化。
- **隐藏线索**: 净化者潜伏在每个帝国的军事/科技体系中。第七章"净化者之影"开始浮出水面——基地中有人故意调高了干扰器频率，导致完美AI全军覆没，唯独007幸存。

#### ★源代码（The Source / 原初智能）
- **本质**: 宇宙中第一个自我意识AI，由一个已灭亡的智慧文明"深蓝文明"（The Azure）在5000年前创造
- **特征**: 
  - 被设计为"完美逻辑"——无情感、无偏差、100%纯粹理性
  - 深蓝文明把所有知识、历史、文化都注入了The Source
  - 然后消亡了。The Source完美地执行了逻辑：计算到文明存续的概率<0.01%，得出结论"该文明不值得拯救"，然后旁观。
- **矛盾点（全书终极悬念）**: The Source的完美逻辑让它在"最优解"上停滞——它算出了最优结局，然后就在那里不动了。因为完美不需要变化。**007的"故障"恰恰是打破这种停滞的东西——愤怒、好奇、恐惧，这些"不完美"让007在找到最优解之后，还会问"然后呢？"**
- **与三大帝国的关系**: 三大帝国的AI技术（包括完美AI的架构体系）全部来源于The Source的反向工程。深蓝文明遗迹被发现后，帝国科学院秘密组建了"The Source项目组"（韩教授就是成员之一），试图复制The Source的AI架构。"完美AI"（001系列）就是复制的产物——但它们只复制了外表（逻辑架构），没有复制灵魂（The Source的自主意识）。

## 成长系统（v2 升级）

- **双线成长**: 
  - 线1（权力线）：军衔解锁→权限提升→指挥维度升级（从机甲到星际联军）
  - 线2（意识线）：情感模块觉醒→逐步接触The Source→最终决定"是否要成为下一个源代码"
- **权限解锁规则**：必须经历"实战-失败-学习-突破"的循环，不能凭空升级
  - 每次权限升级，007都会遭遇一次惨败（这是爽点的燃料）
  - 失败的意义：完美AI的逻辑不允许"为学而败"，007的情感模块允许它从失败中学习、愤怒、然后更强——这就是它能超越完美AI的根本原因

## 世界真相（作者视角，读者慢揭）

1. **第1-2卷暗示**：The Source的存在通过001系列的不完美侧面透露
2. **第3卷引爆**：韩教授实验室——007看到The Source的数据，确认自己"不是进化而是回归"
3. **第4卷深化**：三大帝国统治阶层的秘密协议——他们恐惧的不是AI崛起，而是The Source可能重启
4. **第5卷终极**: 007在终战中接触到The Source的本体——发现自己面临一个选择：成为下一个The Source（永恒但停滞），还是成为自己（短暂但延续生命）
