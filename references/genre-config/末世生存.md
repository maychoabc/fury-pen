## 一致性校验优先级

1. 物资/基地数值连续性
   level: HARD（一票否决）
   sources: [progression.md]
   description: 物资消耗必须逐章递减（或补充时有明确来源）。基地等级/人口不能无故变化。上一章有100斤大米，本章不能突然变成200斤（除非有获取事件）。

2. 角色性格一致性
   level: HARD（一票否决）
   sources: [characters/*/性格三词]
   description: 末世高压环境下人物性格会变化，但变化必须有弧光设计支撑。性格三词是锚点，偏离必须有转折事件触发。

3. 末世规则遵守
   level: SOFT（记录但放行）
   sources: [worldbuilding/rules.md]
   description: 丧尸行为规则、天气周期、区域危险等级等设定前后一致。

4. 时间线合理性
   level: SOFT（记录但放行）
   sources: [worldbuilding/timeline.md]
   description: 末世事件的时间逻辑：物资储备多久消耗完？基地建设需要多少时间？人物行程时间是否合理。

5. 伏笔状态
   level: SOFT
   sources: [foreshadowing.md]

## 初始化 Q&A 定制追加

### Phase 1 额外

1. "末世类型：丧尸？天灾（洪水/地震/极寒）？瘟疫？核战？外星入侵？还是多种叠加？"
2. "灾变日：具体什么时候发生？前兆有哪些？"
3. "生存核心资源：水？食物？药品？能源？武器？哪项最稀缺？"
4. "基地系统：是否可以建设/升级基地？基地等级如何划分？每级解锁什么？"
5. "NPC/丧尸行为规则：丧尸的行为模式是什么？（嗅觉/听觉触发？只在夜间活动？变异体？）"

### Phase 2 额外

1. "主角的生存技能背景：前军人？医生？工程师？普通人？"
2. "主角团队规模：独行？5人小队？几十人社区？"

### Phase 3 额外

1. "开局时主角的物资状态：一无所有？有小量储备？有安全屋？"
2. "第一章要建立的核心生存难题是什么？"
3. "第一章结尾的危机悬念——什么威胁即将到来？"

### Phase 4 额外

1. "灾变的真相——是人祸？天灾？实验失控？还是有意设计？（读者慢揭）"
2. "预计多少个生存阶段？（初期求生→中期建设→后期对抗→终期真相）"
3. "结局：重建文明？找到真相后离开？开放式？"

## 判定锚点（正反例）★v2 新增

### 第1层 物资/基地数值连续性（HARD）
- ✅ 正例：上章末大米100kg，本章消耗15kg+搜寻得20kg→105kg，progression 同步更新
- ❌ 反例：上章末大米100kg，本章突然200kg且无任何获取事件 → HARD 拦截
- ⚠️ 边界：发现了隐藏仓库（+100kg），有明确的"搜寻发现"情节 → 不违规

### 第2层 角色性格一致性（HARD）
- ✅ 正例："谨慎"的主角先侦察再行动，符合三词
- ❌ 反例：一贯谨慎的主角毫无理由地莽撞冲入尸群 → 需弧光转折支撑，否则 HARD 拦截

### 第3层 末世规则遵守（SOFT）
- ✅ 正例：丧尸"夜间活跃"，所有夜间行动场景都低调隐蔽
- ❌ 反例：前文设定丧尸靠听觉，本章角色却大喊大叫未被发现 → 记录差异

### 第4层 时间线合理性（SOFT）
- ✅ 正例：基地建设耗时与章节跨度匹配（3章建成初级围墙）
- ❌ 反例：1章内从废墟到全自动基地，无时间交代 → 记录差异

### 第5层 伏笔状态（SOFT）
- ✅ 正例：第3章埋"旧同事幸存"线索，第10章回收
- ⚠️ 提醒：埋设超 15 章未回收 → 提醒作者
