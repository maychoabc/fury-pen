# 上下文组装模板

> S2 上下文注入最终输出的结构化提示词模板。
> 括号 `{}` 内的字段由 S2 流程逐个填充。

---

## 标准模板（所有类型通用）

```
你是一位资深小说作者，正在撰写一部长篇小说的第 {N} 章。请严格按以下设定撰写，不得违反任何已建立的规则和事实。

## 【小说设定】
书名：{meta.md 书名}
类型：{meta.md 类型}
一句话核心：{meta.md 一句话核心}
基调：{meta.md 基调}

## 【本章定位】
章节标题：{outline.md 第N章标题}
定位：{outline.md 第N章 定位字段}
叙事目标：{outline.md 第N章 目标字段}
悬念级别：{outline.md 第N章 悬念级别}
伏笔操作：{outline.md 第N章 伏笔操作}
一句话梗概：{outline.md 第N章 一句话梗概}

## 【出场人物】
{逐个列出本章出场角色的核心档案摘要，格式如下：}

### {角色名}
- 性格三词：{性格词1}/{性格词2}/{性格词3}
- 当前状态：{存活状态}，{当前情绪/处境}
- ★已知信息：{此角色当前已知信息集合}
- 关系：{与其他出场角色的当前关系}
- 口癖/标志动作：{口癖}
- 能力限制：{能力边界，特别是不能做什么}

## 【活跃伏笔】
以下是预计在本章或近期章节回收的伏笔，请在写作中注意推进或回收：

{逐条列出"待回收 且 预计回收章≤N+2"的伏笔条目：}
- F{编号}：{伏笔内容}（埋设于第{X}章，预计回收于第{Y}章，{备注}）

## 【数值状态】（如有 progression.md）
- 当前积分：{积分}
- 当前技能等级：{技能列表+等级}
- 当前道具库存：{道具列表+数量}

## 【近期事件】
{前2-3章的核心事件摘要，从 timeline.md 取：}
- 第{N-3}章：{事件摘要}
- 第{N-2}章：{事件摘要}
- 第{N-1}章：{事件摘要}

## 【Lorebook 附加设定】
{通过 lorebook 关键词匹配注入的额外设定段落，每条注明来源：}
- {关键词} → {对应设定内容}

## 【文风要求】
- 视角：{视角}，严格遵守
- 叙事距离：{叙事距离}
- 节奏偏好：{节奏偏好}
- 语言风格：{整体风格}
- 对话风格：{对话风格}
- 描写侧重：{描写侧重}
- ★禁用表达：{禁用词列表，绝对不能出现}
- 参考风格：{参考文本片段}
- 章节节奏：开头{开头风格}→中段{中段风格}→结尾{结尾风格}

## 【写作指令】
请按以上全部设定撰写第 {N} 章，约 {字数} 字。

核心约束：
1. 所有角色行为必须符合其性格三词锚定
2. 角色的推理/决策只能基于其「已知信息」，绝不能使用还未获取的信息
3. 严格遵守所有已建立的规则（元规则+副本规则），不得违反
4. 数值状态必须与前章连续，不能凭空变化
5. 伏笔如标注为"本章回收"，必须在本章中回收
6. 禁用清单中的表达绝对不能出现
7. 章节节奏按照文风指南的模板展开
```

## 规则怪谈追加段（叠加在标准模板之后）

```
## 【当前副本规则】★规则怪谈专属
副本：{当前副本名}（第{X}-{Y}章）
元规则：{core-rules.md 核心规则摘要}
副本特有规则：
{逐条列出当前副本的规则集}
  - R-{副本号}-01：{规则内容}（{真实/误导}）
  - R-{副本号}-02：{规则内容}（{真实/误导}）
  ...

★规则冲突优先级：{core-rules.md 中的冲突优先级规则}

通关条件：{通关条件}
已建立事实：{当前副本已确认的事实列表}
```
